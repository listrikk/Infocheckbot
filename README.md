# 🔎Check information bot 🔍

Бот предназначен для проверки достоверности информации, поступающей в виде новостей, слухов или сообщений. Он использует API OpenRouter и модель DeepSeek для определения, является ли новость правдой или фейком.

## 🔧 Функциональность

- Проверка текста с помощью OpenRouter AI
- Кеширование проверок в базе данных SQLite
- Кнопочное меню в Telegram
- Автоматическое удаление старых записей из базы

## 🚀 Установка и запуск

1. Склонируйте репозиторий:

```bash
git clone https://github.com/listrikk/Infocheckbot.git
cd Infocheckbot
```

2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Откройте файл `botcheck.py` и вставьте свои токены:

```python
TELEGRAM_TOKEN = 'ваш_токен_бота'
OPENROUTER_API_KEY = 'ваш_API_ключ'
```

4. Запустите бота:

```bash
python botcheck.py
```

## 🗃 Структура проекта

- `botcheck.py` — основной файл с кодом Telegram-бота
- `news.db` — создаётся автоматически, содержит проверенные новости
- `requirements.txt` — список зависимостей
- `README.md` — текущая инструкция

## 📌 Примечания

- SQLite используется для хранения уже проверенных новостей, чтобы не перегружать API
- Если записей становится больше 100, старые автоматически удаляются
